BCD-UI
Overview

BCD-UI is a full-stack framework for building enterprise applications with web technologies.

BCD-UI features a set of frequently used components for reporting and data management and is built on today's technologies.

BusinessCode

The company behind BCD-UI is BusinessCode GmbH, Germany.

BCD-UI can be used as a part of BusinessCode's BCD-Suite, which also provides features for loading, cleaning, calculating and enriching data in you warehouse with support of a graphical logic editor BCD-BM.

For more information on BCD-Suite visit https://www.business-code.de.
